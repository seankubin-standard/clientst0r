Acme Corporation Demo Data

This will automatically create (or use existing) "Acme Corporation" organization and populate it with complete demo data including documents, diagrams, assets, passwords, KB articles, and processes.

What will be imported:
  • Organization: Creates "Acme Corporation" (if it doesn't exist)
  • Documents: IT Procedures, Security Policies, and Runbooks
  • Diagrams: Network and Rack Diagrams
  • Assets: Workstations, Servers, and Network Equipment
  • Passwords: Sample Passwords (demo credentials)
  • KB Articles: Knowledge Base Articles
  • Processes: IT Processes with execution history
Note: You will automatically be added as an admin to the "Acme Corporation" organization. You can switch to it from the organization dropdown in the navbar.
