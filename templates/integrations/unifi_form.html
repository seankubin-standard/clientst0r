Self-hosted Setup
  1. Log into UniFi OS (your controller URL)
  2. Go to Settings → Control Plane → Integrations → API
  3. Click Create API Key and copy it
  4. Optional: add a local admin username/password for WLAN and VLAN data
Cloud (Site Manager) Setup
  1. Go to account.ui.com → API Keys
  2. Click Generate API Key and copy it
  3. Select Cloud (UniFi Site Manager) as the mode
  4. Leave the host URL blank — the cloud API URL is used automatically
Cloud mode retrieves sites and devices across all your UI.com-managed hosts. WLAN/VLAN/firewall data is not available via the Site Manager API.
