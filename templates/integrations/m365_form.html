Setup Guide
  1. Go to Azure portal → Azure Active Directory → App registrations → New registration
  2. Copy the Directory (tenant) ID and Application (client) ID
  3. Go to Certificates & secrets → New client secret
  4. Under API permissions, add these Application permissions:
    • User.Read.All
    • Group.Read.All
    • Sites.Read.All
    • Directory.Read.All
    • Organization.Read.All
    • SecurityAlert.Read.All (for Defender alerts)
    • TeamSettings.Read.All (for Teams data)
    • Reports.Read.All (for usage reports)
  5. Click Grant admin consent
